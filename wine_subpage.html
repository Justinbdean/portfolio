Wine Explosion

Houdini, Karma, Solaris

2024

This project was built ontop of a SideFX tutorial that I watched over the summer. I decided as a challenge to come back to it in order to learn Solaris and lighting within Houdini. In order to better communicate with artists and other creatives, I've put more time into learning what efforts are made to create 3D graphics. This project was a great stepping stone in understanding the various considerations of how to create a scene and what challenges can be presented when working within Houdini.
